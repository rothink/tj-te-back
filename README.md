# Passo a passo back - report

- Clonar repositório
  >
        git clone https://github.com/rothink/tj-te-back.git


- Entrar no repositório clonado
  >
        cd tj-te-back


- Subir os serviços com docker e instalar as dependências
  >
        make install


- Rodas as migrations
  >
        make migration

- Rodas as migrations com seeder
  >
        make migration

- Executar os testes
  >
        make test
